# bid-backend

## Run

From this folder:

```bash
mvn spring-boot:run
```

Backend runs at `http://localhost:8080`.

- H2 console: `http://localhost:8080/h2`
- Uploaded images are served at: `http://localhost:8080/uploads/<filename>`

## API

- `GET /api/products`
- `GET /api/products/{id}`
- `POST /api/products` (multipart/form-data)
- `PUT /api/products/{id}` (multipart/form-data)
- `DELETE /api/products/{id}`
- `POST /api/products/{id}/bids` (JSON)
