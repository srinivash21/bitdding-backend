# Running (Backend)

## Prerequisites
- Windows 10/11
- JDK installed (this project was run with Java 23)

## Run
From this folder, run:
- `RUN_BACKEND.cmd`

Or manually:
- `./.tools/apache-maven-3.9.12/bin/mvn.cmd spring-boot:run`

## URLs
- API: http://localhost:8080/
- H2 console (if enabled): http://localhost:8080/h2

## Stop
Press `Ctrl+C` in the backend terminal.
